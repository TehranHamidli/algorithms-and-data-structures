Given a natural number N, find all twin prime numbers not exceeding N (N < 10000). Twin primes are prime numbers that differ from each other by 2.

Input Format

The input consists of a single line containing a natural number N.

Constraints

2 <= N < 10000

Output Format

Print all twin prime numbers not exceeding N in ascending order.

Sample Input 0

50
Sample Output 0

(3, 5)
(5, 7)
(11, 13)
(17, 19)
(29, 31)
(41, 43)
Sample Input 1

150
Sample Output 1

(3, 5)
(5, 7)
(11, 13)
(17, 19)
(29, 31)
(41, 43)
(59, 61)
(71, 73)
(101, 103)
(107, 109)
(137, 139)
