Given two natural numbers A and B, find their least common multiple (LCM).

Input Format

The input consists of two lines containing two natural numbers A and B.

Constraints

1 <= A, B <= 10^9

Output Format

Print the LCM of A and B as a single integer.

Sample Input 0

6
8
Sample Output 0

24
Sample Input 1

12
18
Sample Output 1

36
Sample Input 2

12
24
Sample Output 2

24
