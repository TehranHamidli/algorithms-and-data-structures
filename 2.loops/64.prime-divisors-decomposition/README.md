Given a natural number N, decompose it into its prime divisors.

Input Format

The input consists of a single line containing a natural number N.

Constraints

1 <= N <= 10^9

Output Format

Print the prime divisors of N in ascending order, along with their powers as exponents.

Sample Input 0

60
Sample Output 0

2^2 3^1 5^1
Explanation 0

The given number is N = 60. The prime divisors of 60 are 2, 3, and 5. The prime factorization of 60 is 2^2 * 3^1 * 5^1, so the output is "2^2 3^1 5^1".
