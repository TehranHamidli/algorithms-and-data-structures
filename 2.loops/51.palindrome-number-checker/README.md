Given a natural number N, determine if it is a palindrome (reads the same forwards and backwards).

Input Format

The input consists of a single line containing a natural number N.

Constraints

The value of N will be within the range of a natural number.

Output Format

Print "YES" if the number is a palindrome, otherwise print "NO".

Sample Input 0

12321
Sample Output 0

YES
Sample Input 1

100
Sample Output 1

NO
Sample Input 2

12345
Sample Output 2

NO
Sample Input 3

1223221
Sample Output 3

YES
Sample Input 4

12233226
Sample Output 4

NO
