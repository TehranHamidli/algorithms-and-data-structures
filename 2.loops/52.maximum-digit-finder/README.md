Given a natural number N, find the maximum digit present in this number.

Input Format

The input consists of a single line containing a natural number N.

Constraints

The value of N will be within the range of a natural number.

Output Format

Print the maximum digit as a single integer.

Sample Input 0

567891
Sample Output 0

9
Sample Input 1

12345
Sample Output 1

5
Sample Input 2

11111
Sample Output 2

1
