Given a natural number N, determine if it is a power of 2.

Input Format

The input consists of a single line containing a natural number N.

Constraints

1 <= N <= 10^9

Output Format

Print "Yes" if N is a power of 2, otherwise print "No".

Sample Input 0

8
Sample Output 0

Yes
Sample Input 1

128
Sample Output 1

Yes
Sample Input 2

1
Sample Output 2

Yes
Sample Input 3

12
Sample Output 3

No
