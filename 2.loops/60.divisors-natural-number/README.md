Given a natural number N, find all its divisors.

Input Format

The input consists of a single line containing a natural number N.

Constraints

1 <= N <= 10^9

Output Format

Print all the divisors of N in ascending order.

Sample Input 0

6
Sample Output 0

1 2 3 6
Explanation 0
a
Explanation:

The given number is N = 6. The divisors of 6 are 1, 2, 3, and 6.

Sample Input 1

24
Sample Output 1

1 2 3 4 6 8 12 24
