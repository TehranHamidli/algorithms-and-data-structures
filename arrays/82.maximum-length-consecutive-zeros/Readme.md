Given a sequence of integers, find the length of the section with the maximum number of consecutive zeros.

Input Format

The first line contains an integer N, representing the size of the sequence. The second line contains N space-separated integers representing the elements of the sequence.

Constraints

1 <= N <= 100 Each integer in the sequence will be within the range of -1000 to 1000.

Output Format

Print the length of the section with the maximum number of consecutive zeros.

Sample Input 0

10
1 0 0 2 0 0 0 3 0 4
Sample Output 0

Length of Maximum Consecutive Zeros: 3
Sample Input 1

8
0 0 0 0 1 2 3 0
Sample Output 1

Length of Maximum Consecutive Zeros: 4
Sample Input 2

12
1 0 2 0 0 3 4 2 1 0 0 0
Sample Output 2

Length of Maximum Consecutive Zeros: 3
