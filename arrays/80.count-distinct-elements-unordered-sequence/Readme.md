Given an array of integers, find the number of distinct elements in the array (ignoring duplicate occurrences).

Input Format

The first line contains an integer N, representing the size of the array. The second line contains N space-separated integers representing the elements of the array.

Constraints

1 <= N <= 100 Each integer in the array will be within the range of -1000 to 1000.

Output Format

Print the number of distinct elements in the array.

Sample Input 0

8
2 5 3 8 2 4 8 7
Sample Output 0

Distinct Elements: 6
Sample Input 1

6
-1 -3 -1 0 0 3
Sample Output 1

Distinct Elements: 4
Sample Input 2

16
3 2 0 3 8 9 2 4 3 4 5 6 3 6 7 0
Sample Output 2

Distinct Elements: 9
Sample Input 3

5
0 0 0 0 0
Sample Output 3

Distinct Elements: 1
