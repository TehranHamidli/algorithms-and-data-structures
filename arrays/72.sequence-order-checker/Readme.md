Given a sequence of integers, check if it is in ascending order, descending order, or neither. An ascending sequence is a sequence in which each element is greater than to the previous element. A descending sequence is a sequence in which each element is less than to the previous element. If none of these conditions are met, the sequence is neither ascending nor descending.

Input Format

The input consists of a single line containing space-separated integers representing the sequence.

Constraints

The sequence will contain at least two integers. Each integer in the sequence will be within the range of -1000 to 1000.

Output Format

Print "Ascending" if the sequence is in ascending order, "Descending" if it is in descending order, or "Neither" if it does not fit either of these patterns.

Sample Input 0

5
1 3 5 7 9
Sample Output 0

Ascending
Sample Input 1

5
10 8 6 4 2
Sample Output 1

Descending
Sample Input 2

5
2 5 1 9 4
Sample Output 2

Neither
