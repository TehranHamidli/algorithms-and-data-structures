Given a non-decreasing sequence of integers, find the number of distinct elements in the sequence.

Input Format

The first line contains an integer N, representing the size of the sequence. The second line contains N space-separated integers representing the elements of the sequence.

Constraints

1 <= N <= 100 Each integer in the sequence will be within the range of -1000 to 1000.

Output Format

Print the number of distinct elements in the sequence.

Sample Input 0

7
2 4 4 6 8 8 8
Sample Output 0

Distinct Elements: 4
Explanation 0

Explanation:

The given non-decreasing sequence is 2 4 4 6 8 8 8. The distinct elements in the sequence are 2, 4, 6, and 8.
