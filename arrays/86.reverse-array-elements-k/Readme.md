Given an array of N elements and an integer K (1 <= K <= N), reverse the elements of the array from index 1 to index K.

Input Format

The first line contains an integer N, representing the size of the array. The second line contains N space-separated integers representing the elements of the array. The third line contains an integer K.

Constraints

1 <= N <= 100 1 <= K <= N

Output Format

Print the modified array after reversing elements from index 1 to index K. Separate the elements by space.

Sample Input 0

5
1 2 3 4 5
3
Sample Output 0

3 2 1 4 5
Sample Input 1

8
10 20 30 40 50 60 70 80
6
Sample Output 1

60 50 40 30 20 10 70 80
Sample Input 2

8
10 20 30 40 50 60 70 80
8
Sample Output 2

80 70 60 50 40 30 20 10
