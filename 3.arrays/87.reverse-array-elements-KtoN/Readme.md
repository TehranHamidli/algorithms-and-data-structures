Given an array of N elements and an integer K (1 <= K <= N), reverse the elements of the array from index K to index N.

Input Format

The first line contains an integer N, representing the size of the array. The second line contains N space-separated integers representing the elements of the array. The third line contains an integer K.

Constraints

1 <= N <= 100 1 <= K <= N

Output Format

Print the modified array after reversing elements from index K to index N. Separate the elements by space.

Sample Input 0

5
1 2 3 4 5
3
Sample Output 0

1 2 5 4 3
Sample Input 1

8
10 20 30 40 50 60 70 80
5
Sample Output 1

10 20 30 40 80 70 60 50
Sample Input 2

8
10 20 30 40 50 60 70 80
1
Sample Output 2

80 70 60 50 40 30 20 10
