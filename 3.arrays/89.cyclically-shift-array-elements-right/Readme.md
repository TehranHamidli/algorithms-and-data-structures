Given an array of N elements, cyclically shift the array elements by one position to the right.

Input Format

Given an array of N elements, cyclically shift the array elements by one position to the right.

Constraints

1 <= N <= 100

Output Format

Print the modified array after cyclically shifting the elements to the right. Separate the elements by space.

Sample Input 0

6
1 2 3 4 5 6
Sample Output 0

6 1 2 3 4 5
Sample Input 1

5
10 20 30 40 50
Sample Output 1

50 10 20 30 40
Sample Input 2

2
50 10
Sample Output 2

10 50
