Given a sequence of integers, check if it forms a geometric progression. A geometric progression is a sequence of numbers in which the ratio between any two consecutive terms is constant.

Input Format

The input consists of a single line containing space-separated integers representing the sequence.

Constraints

The sequence will contain at least two integers. Each integer in the sequence will be within the range of -1000 to 1000. The ratio between consecutive terms in the sequence will not be zero.

Output Format

Print "Yes" if the sequence is a geometric progression, otherwise print "No".

Sample Input 0

5
2 4 8 16 32
Sample Output 0

Yes
Sample Input 1

5
3 9 27 81 243
Sample Output 1

Yes
Sample Input 2

5
1 2 5 10 20
Sample Output 2

No
