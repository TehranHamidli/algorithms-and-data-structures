Given a sequence of integers, check if it forms an arithmetic progression. An arithmetic progression is a sequence of numbers in which the difference between any two consecutive terms is constant.

Input Format

The input consists of a single line containing space-separated integers representing the sequence.

Constraints

The sequence will contain at least two integers. Each integer in the sequence will be within the range of -1000 to 1000.

Output Format

Print "Yes" if the sequence is an arithmetic progression, otherwise print "No".

Sample Input 0

5
1 3 5 7 9
Sample Output 0

Yes
Sample Input 1

6
10 20 30 40 50 60
Sample Output 1

Yes
Sample Input 2

5
1 2 4 8 16
Sample Output 2

No
