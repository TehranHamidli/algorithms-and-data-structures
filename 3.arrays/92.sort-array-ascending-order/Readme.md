Given an array of N elements, sort the elements in ascending order.

Input Format

The first line contains an integer N, representing the size of the array. The second line contains N space-separated integers representing the elements of the array.

Constraints

1 <= N <= 100 Each element of the array is an integer.

Output Format

Print the sorted array in ascending order. Separate the elements by space.

Sample Input 0

5
3 1 4 2 5
Sample Output 0

1 2 3 4 5
Sample Input 1

7
9 6 8 3 2 1 4
Sample Output 1

1 2 3 4 6 8 9
Sample Input 2

13
2 3 2 -1 0 8 0 -2 5 -2 1 1 0
Sample Output 2

-2 -2 -1 0 0 0 1 1 2 2 3 5 8
