Given an array of integers, find the maximum and minimum elements in the array and then swap their positions.

Input Format

The first line contains an integer N, representing the size of the array. The second line contains N space-separated integers representing the elements of the array.

Constraints

1 <= N <= 100 Each integer in the array will be within the range of -1000 to 1000.

Output Format

Print the array after swapping the positions of the maximum and minimum elements.

Sample Input 0

6
8 3 10 5 2 7
Sample Output 0

8 3 2 5 10 7
Sample Input 1

5
-6 0 12 4 -2
Sample Output 1

12 0 -6 4 -2
