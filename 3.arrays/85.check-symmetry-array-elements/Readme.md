Given an array of integers, check if the values of array elements are symmetrical. An array is considered symmetrical if it reads the same from left to right and from right to left.

Input Format

The first line contains an integer N, representing the size of the array. The second line contains N space-separated integers representing the elements of the array.

Constraints

1 <= N <= 100 Each integer in the array will be within the range of -1000 to 1000.

Output Format

If the values of array elements are symmetrical, print "Symmetrical". If the values of array elements are not symmetrical, print "Not Symmetrical".

Sample Input 0

5
1 2 3 2 1
Sample Output 0

Symmetrical
Sample Input 1

6
1 2 3 4 5 6
Sample Output 1

Not Symmetrical
