Given a string, write a C++ program to count the number of vowels (a, e, i, o, u) in the string and print the count.

Input Format

A single line containing a string.

Constraints

The input string consists of lowercase and/or uppercase alphabetic characters. The input string has at most length 1000 characters.

Output Format

The program should output the count of vowels in the string in the following format: "number of vowels -> count"

Sample Input 0

eagerer
Sample Output 0

number of vowels -> 4
Sample Input 1

Hello World
Sample Output 1

number of vowels -> 3
Sample Input 2

cppProgramming
Sample Output 2

number of vowels -> 3
