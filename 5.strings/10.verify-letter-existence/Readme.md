Given two strings, write a C++ program to determine whether all the letters in the second string appear in the first string. Return true if they do, otherwise return false.

Input Format

Two strings, each on a separate line.

Constraints

Both strings have at most length 1000. Strings can contain both uppercase and lowercase letters.

Output Format

The program should output "true" if all letters in the second string are found in the first string, and "false" otherwise.

Sample Input 0

CPP
Cpp
Sample Output 0

false
Sample Input 1

Java
Ja
Sample Output 1

true
Sample Input 2

Check first string
sifC
Sample Output 2

true
