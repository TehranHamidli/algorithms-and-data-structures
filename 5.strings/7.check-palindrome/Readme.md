Given a string, write a C++ program to determine whether it is a palindrome. A palindrome is a word, number, phrase, or sequence of characters that reads the same backward as forward.

Input Format

A single line containing a string.

Constraints

The input string consists of lowercase and/or uppercase alphabetic characters. The input string has at most length 1000 characters.

Output Format

The program should output "True" if the input string is a palindrome, and "False" otherwise.

Sample Input 0

madam
Sample Output 0

True
Sample Input 1

racecar
Sample Output 1

True
Sample Input 2

hello
Sample Output 2

False
