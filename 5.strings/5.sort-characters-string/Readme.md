Given a string containing alphabetic characters, write a C++ program to sort and output the characters in the string in ascending order. Exclude numbers and punctuation symbols from sorting.

Input Format

A single line containing a string of alphabetic characters, numbers, and punctuation symbols.

Constraints

The input string contains only alphanumeric characters and punctuation symbols. The input string has at most length 1000 characters.

Output Format

The program should output the sorted alphabetic characters from the input string.

Sample Input 0

python
Sample Output 0

hnopty
Sample Input 1

face123
Sample Output 1

acef
