Write a C++ program that takes a string as input and outputs the reverse of that string.

Input Format

A single line containing a string of length 1 to 1000 characters.

Constraints

None

Output Format

A single line containing the reversed string.

Sample Input 0

Hello
Sample Output 0

olleH
Sample Input 1

World!
Sample Output 1

!dlroW
Sample Input 2

Hello World!
Sample Output 2

!dlroW olleH
