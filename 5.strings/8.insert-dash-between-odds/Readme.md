Given a string of numbers, write a C++ program to insert a dash character ('-') between two distinct odd numbers in the string.

Input Format

A single line containing a string of numbers.

Constraints

The input string consists of digits (0-9). The input string has at most length 1000 characters.

Output Format

The program should output the modified string with dashes inserted between consecutive odd numbers.

Sample Input 0

1345789
Sample Output 0

1-345-789
Sample Input 1

24682468
Sample Output 1

24682468
Sample Input 2

13579
Sample Output 2

1-3-5-7-9
Sample Input 3

135799
Sample Output 3

1-3-5-7-99
