Given a string containing words separated by spaces, write a C++ program to capitalize the first letter of each word while converting the rest of each word to lowercase. Words are separated by a single space.

Input Format

A single line containing a string of words separated by spaces.

Constraints

The input string contains only alphabetical characters and spaces. Words are separated by a single space. The input string has at most length 1000 characters. Words consist of lowercase and uppercase alphabetical characters.

Output Format

The program should output the input string with the first letter of each word capitalized and the rest of each word in lowercase.

cpp string exercises
Sample Output 0

Cpp String Exercises

hello WoRLd
Sample Output 1

Hello World

code </> star is amazing
Sample Output 2
