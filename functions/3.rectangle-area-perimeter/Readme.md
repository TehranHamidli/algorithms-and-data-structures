You are given the length and width of a rectangle. Write a C++ program to calculate its area and perimeter.

Input Format

Two integers, length and width, separated by spaces.

Constraints

1 <= length, width <= 100

Output Format

The area of the rectangle. The perimeter of the rectangle.

Sample Input 0

5 8
Sample Output 0

The area of the rectangle is: 40
The perimeter of the rectangle is: 26
Sample Input 1

10 15
Sample Output 1

The area of the rectangle is: 150
The perimeter of the rectangle is: 50
Sample Input 2

3 3
Sample Output 2

The area of the rectangle is: 9
The perimeter of the rectangle is: 12
