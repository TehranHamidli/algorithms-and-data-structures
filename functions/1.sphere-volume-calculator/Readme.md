Write a C++ function that calculates the volume of a sphere. Given the radius as input, the function should output the volume of the sphere. The formula to calculate the volume of a sphere is (4/3) * π * r^3, where "r" is the radius of the sphere.

Input Format

A single floating-point number representing the radius of the sphere.

Constraints

The radius is a positive real number.

Output Format

Print the calculated volume of the sphere.

Sample Input 0

6
Sample Output 0

The volume of a sphere is: 904.779
Sample Input 1

3.5
Sample Output 1

The volume of a sphere is: 179.594
Sample Input 2

10
Sample Output 2

The volume of a sphere is: 4188.79
