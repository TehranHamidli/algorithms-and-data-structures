Write a C++ program that takes a decimal number as input and converts it to a binary number using a function. Display the binary value.

Input Format

A single integer representing the decimal number.

Constraints

The input decimal number should be non-negative.

Output Format

A message indicating the binary value of the entered decimal number.

Sample Input 0

65
Sample Output 0

1000001
Sample Input 1

12
Sample Output 1

1100
Sample Input 2

255
Sample Output 2

11111111
