Write a C++ function that calculates the volume of a cube and a program that utilizes this function. The user will input the side length of the cube, and your program should output the volume.

double calculateCubeVolume(double sideLength)

Input Format

A single integer representing the side length of the cube.

Constraints

1 <= sideLength <= 100

Output Format

A single floating-point number representing the volume of the cube.

Sample Input 0

5
Sample Output 0

125
Sample Input 1

3
Sample Output 1

27
Sample Input 2

10
Sample Output 2

1000
