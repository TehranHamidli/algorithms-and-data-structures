Write a C++ program that takes a positive number as input and checks whether it is a prime number using a function. Display the result.

Input Format

A single positive integer.

Constraints

The input number should be a positive integer.

Output Format

A message indicating whether the entered number is a prime number or not.

Sample Input 0

5
Sample Output 0

The number 5 is a prime number
Sample Input 1

12
Sample Output 1

The number 12 is not a prime number
Sample Input 2

17
Sample Output 2

The number 17 is a prime number
