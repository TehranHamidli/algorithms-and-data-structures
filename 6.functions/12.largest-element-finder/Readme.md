Write a C++ program that takes an array of integers as input and finds the largest element in the array using a function. Display the result.

Input Format

An array of integers. The length of the array 'n' (1 <= n <= 100). n integers separated by spaces.

Constraints

Each integer in the array is within the range [-1000, 1000].

Output Format

A message indicating the largest element in the array.

Sample Input 0

9
1 2 3 4 10 5 6 7 9
Sample Output 0

10
Sample Input 1

5
-5 -10 -1 -3 -2
Sample Output 1

-1
Sample Input 2

3
7 7 7
Sample Output 2

7
