Write a C++ program that takes a positive integer as input and checks whether it is an Armstrong number and whether it is a Perfect number using functions. Display the results for both checks.

Input Format

A single positive integer.

Constraints

none

Output Format

Two messages indicating whether the entered number is an Armstrong number and whether it is a Perfect number.

Sample Input 0

371
Sample Output 0

The 371 is an Armstrong number.
The 371 is not a Perfect number.
Sample Input 1

28
Sample Output 1

The 28 is not an Armstrong number.
The 28 is a Perfect number.
Sample Input 2

1634
Sample Output 2

The 1634 is an Armstrong number.
The 1634 is not a Perfect number.
