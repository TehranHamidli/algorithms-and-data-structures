Write a C++ program that takes a range as input and finds all the perfect numbers within that range using a function. Display the list of perfect numbers.

Input Format

Two integers representing the range: "from" and "to."

Constraints

none

Output Format

A message indicating the perfect numbers within the specified range.

Sample Input 0

1 100
Sample Output 0

The perfect numbers between 1 to 100 are: 6 28
Sample Input 1

 1 50
Sample Output 1

The perfect numbers between 1 to 50 are: 6 28
Sample Input 2

1 10000
Sample Output 2

The perfect numbers between 1 to 10000 are: 6 28 496 8128
