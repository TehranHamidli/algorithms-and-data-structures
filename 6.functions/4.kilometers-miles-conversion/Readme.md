A single floating-point number representing the speed in kilometers per hour.

Input Format

A message indicating the conversion result with 4 decimal places.

Constraints

0 <= speed <= 1000

Output Format

A message indicating the conversion result with 4 decimal places.

Sample Input 0

25
Sample Output 0

15.5343 Miles/hr
Sample Input 1

50
Sample Output 1

31.0686 Miles/hr
Sample Input 2

10.5
Sample Output 2

6.5244 Miles/hr
