Write a C++ function to calculate the total and average of four numbers. Divide the total by 4 to find the average.

Input Format

Two pairs of integers, each separated by a space. The first pair represents the first two numbers, and the second pair represents the last two numbers.

Constraints

none

Output Format

Two lines: The total of four numbers. The average of four numbers with two decimal places.

Sample Input 0

25 20
15 25
Sample Output 0

The total of four numbers is : 85
The average of four numbers is : 21.25
Sample Input 1

10 5
30 40
Sample Output 1

The total of four numbers is : 85
The average of four numbers is : 21.25
Sample Input 2

0 0
0 0
Sample Output 2

The total of four numbers is : 0
The average of four numbers is : 0
