Write a C++ program that takes a number as input and checks whether it's even or odd using a function. Display the result.

Input Format

A single integer.

Constraints

A message indicating whether the entered number is even or odd.

Output Format

A message indicating whether the entered number is even or odd.

Sample Input 0

5
Sample Output 0

The entered number is odd
Sample Input 1

6
Sample Output 1

The entered number is even
