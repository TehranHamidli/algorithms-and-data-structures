Write a C++ program that takes two strings as input and checks whether they are anagrams of each other using a function. Display the result.

Input Format

Two strings, separated by newlines.

Constraints

none

Output Format

A message indicating whether the two strings are anagrams or not.

Sample Input 0

anagram
nagaram
Sample Output 0

Anagram
Sample Input 1

DUSSTY
STUDYY
Sample Output 1

Not Anagram
Sample Input 2

dear
eabr
Sample Output 2

Not Anagram
Sample Input 3

spare
pears
Sample Output 3

Anagram
Sample Input 4

listen
silent
Sample Output 4

Anagram
