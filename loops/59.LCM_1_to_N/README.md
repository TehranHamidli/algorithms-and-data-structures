Given a natural number N, find the least common multiple (LCM) of all integers from 1 to N.

Input Format

The input consists of a single line containing a natural number N.

Constraints

1 <= N <= 100

Output Format

Print the LCM of all integers from 1 to N as a single integer.

Sample Input 0

5
Sample Output 0

60
Explanation 0

Explanation:

The given number is N = 5. The least common multiple (LCM) of all integers from 1 to 5 (1, 2, 3, 4, 5) is 60.
