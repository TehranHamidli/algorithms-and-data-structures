Given a natural number N, count the number of digits in that number.

Input Format

The input consists of a single line containing a natural number N.

Constraints

The value of N will be within the range of a natural number.

Output Format

Print the count of digits in N as a single integer.

Sample Input 0

987654321
Sample Output 0

9
Explanation 0

Explanation:

The given number is 987654321. It has 9 digits. The program should output 9.

Sample Input 1

3
Sample Output 1

1
