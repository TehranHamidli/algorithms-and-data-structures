Given a natural number N, determine if it is a prime number.

Input Format

The input consists of a single line containing a natural number N.

Constraints

1 <= N <= 10^9

Output Format

Print "Yes" if N is a prime number, otherwise print "No".

Sample Input 0

5
Sample Output 0

Yes
Sample Input 1

18
Sample Output 1

No
Sample Input 2

17
Sample Output 2

Yes
