Given a natural number N, expand N! (N factorial) into its prime divisors.

Input Format

The input consists of a single line containing a natural number N.

Constraints

1 <= N <= 100

Output Format

Print the prime divisors of N! in ascending order, along with their powers as exponents.

Sample Input 0

5
Sample Output 0

2^3 3^1 5^1
Explanation 0

Explanation:

The given number is N = 5. The factorial of 5 (5!) is 120. The prime divisors of 120 are 2, 3, and 5. The prime factorization of 120 is 2^3 * 3^1 * 5^1, so the output is "2^3 3^1 5^1".
