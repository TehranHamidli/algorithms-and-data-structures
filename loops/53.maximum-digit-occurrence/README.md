Given a natural number N, find out how many times its maximum digit occurs in this number.

Input Format

The input consists of a single line containing a natural number N.

Constraints

The value of N will be within the range of a natural number.

Output Format

Print the count of occurrences of the maximum digit as a single integer.

Sample Input 0

673757
Sample Output 0

3
Sample Input 1

4935
Sample Output 1

1
Sample Input 2

11111
Sample Output 2

5
Sample Input 3

12345
Sample Output 3

1
Sample Input 4

322
Sample Output 4

1
