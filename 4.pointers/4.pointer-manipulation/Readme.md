Consider the following C++ code snippet:

int *p;
int i;
int k;
i = 42;
k = i;
p = &i;
After these statements, which of the following statements will change the value of i to 75?

A) k = 75;
B) *k = 75;
C) p = 75;
D) *p = 75;
Select the correct option(s):

A) Only A
B) Only B
C) Only C
D) Only D
E) Both B and D
F) None of the above
Input Format

No input

Constraints

No constraints

Output Format

Print the correct option: A, B, C, D, E or F
