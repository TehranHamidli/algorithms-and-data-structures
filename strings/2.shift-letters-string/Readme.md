Write a C++ program that takes a string as input and shifts every letter in the string to the next letter in the alphabet while preserving the case (uppercase remains uppercase, and lowercase remains lowercase). Non-alphabetic characters should remain unchanged.

Input Format

A single line containing a string of length 1 to 1000 characters.

Constraints

None

Output Format

A single line containing the modified string.

Sample Input 0

Hello, World!
Sample Output 0

Ifmmp, Xpsme!
Explanation 0

e.g., 'a' becomes 'b', 'p' becomes 'q', 'z' becomes 'a'.

Sample Input 1

abcXYZ123
Sample Output 1

bcdYZA123
