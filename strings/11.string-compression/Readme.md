126. String Compression
Problem
Submissions
Leaderboard
Discussions
Write a function that takes a string as input and compresses it by replacing consecutive repeated characters with the character followed by the count of its repetitions. For example, "aaabbbcc" would become "a3b3c2."

Input Format

A single string containing lowercase letters.

Constraints

The input string contains at most 1000 characters. The input string consists of lowercase letters.

Output Format

Print the compressed string.

Sample Input 0

aaabbbcc
Sample Output 0

a3b3c2
Sample Input 1

aabbaa
Sample Output 1

a2b2a2
Sample Input 2

abcdef
Sample Output 2

a1b1c1d1e1f1
