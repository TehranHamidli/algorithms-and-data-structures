Given a string containing words separated by spaces, write a C++ program to find and output the largest word in the string. The largest word is the one with the most characters. If there are multiple words with the same maximum length, output the first one encountered.

Input Format

A single line containing a string of words separated by spaces.

Constraints

The input string contains only alphabetical characters and spaces. Words are separated by a single space. The input string has at most length 1000 characters. Words consist of lowercase and uppercase alphabetical characters.

Output Format

The program should output the largest word found in the input string.

Sample Input 0

C++ da corek yoxdur.
Sample Output 0

yoxdur
Sample Input 1

The quick brown fox jumps over the lazy elephant.
Sample Output 1

elephant
Sample Input 2

General artificial intelligence is coming.
Sample Output 2

intelligence
