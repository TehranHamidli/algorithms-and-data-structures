124. Remove Special Characters

Submissions

Discussions
Given a string containing alphanumeric characters and special characters, write a C++ program to remove all the special characters from the string.

Input Format

A single line containing a string with alphanumeric characters and special characters.

Constraints

The input string has a length of at most 1000 characters.

Output Format

The program should output the modified string with all special characters removed.

Sample Input 0

abcd $ js# @acde$
Sample Output 0

abcd  js acde
Sample Input 1

hello!world
Sample Output 1

helloworld
Sample Input 2

12345
Sample Output 2

12345
Sample Input 3

abcd!!!$ js# ->@acde$
Sample Output 3

abcd js acde
