You are given two strings as input. Your task is to find the longest substring that is common to all of the given strings. This problem has applications in DNA sequence alignment and plagiarism detection.

Input Format

Multiple strings, each on a separate line. There can be two or more strings.

Constraints

The total length of all strings combined is at most 1000 characters. Strings consist of lowercase and/or uppercase letters.

Output Format

Print the longest common substring among the input strings.

Sample Input 0

abcdxyz
xyzabcd
Sample Output 0

Longest Common Substring: abcd
Sample Input 1

programming
programmer
Sample Output 1

Longest Common Substring: programm
Sample Input 2

apple
banana
Sample Output 2

Longest Common Substring: a
Sample Input 3

aaaabbbb
bbbbb
Sample Output 3

Longest Common Substring: bbbb
